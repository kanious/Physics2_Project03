1. How to build the project

- Please set "Physics_Project03" project as a starter project and build with x64 configuration / Debug or Release mode. Or you can execute with "Physics_Project03.exe" file in x64\Debug(or Release) folder.

- GitHub Link:
https://github.com/kanious/Physics2_Project03



2. User Input Option

< Targeting >
* Tab : Switch target between Vehicle and Character

< Moving >
* WASD : Move target. 
* Space : If target is the vehicle, brake it. If target is the character, jump it.
* F2 : Reset all objects.
* F3 : Apply random force to all Rigidbodies.
* F4 : Add Sphere object

< Looking >
* Move Mouse : Rotate camera angle by Y axis.
* Roll Mouse scroll : Zoom In/Out.

< Etc >
* ESC : Close the program
